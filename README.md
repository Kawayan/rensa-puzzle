# カラーパズル (7×7)

7×7 マスのカラーパネルを動かして、同じ色をつなげて消すパズルゲームです。

## 遊び方

- パネルを **ドラッグ** して自由に動かせます（通過したパネルと順番に入れ替わります）。
- 同じ色のパネルを **上下左右に5個以上** つなげると、徐々に薄くなって **3秒後に消えます**。
- 消えた場所には上から新しいパネルが降ってきます。
- 消えかけのかたまりに、後から同じ色をつなぐと **消える時間がリセット** されます。

## セットアップ / 起動

```bash
npm install      # 依存関係のインストール
npm run build    # TypeScript を dist/app.js にコンパイル
npm start        # lite-server で起動(ブラウザが開きます)
```

開発中は `npm run watch` で TypeScript の自動コンパイルができます。

## 技術スタック

- TypeScript（`src/app.ts` → `dist/app.js`）
- 素の HTML / CSS（フレームワークなし、DOM ベース描画）
- lite-server（開発サーバ）
