# カラーパズル (8×8)

8×8 マスのカラーパネルを動かして、同じ色をつなげて消すパズルゲームです。

## 遊び方

- パネルを **ドラッグ** して自由に動かせます（通過したパネルと順番に入れ替わります）。
- 同じ色のパネルを **上下左右に5個以上** つなげると、色が上から徐々に抜けて **2.5秒後に消えます**。
- 消えた場所には上から新しいパネルが降ってきます。上の行には次に降ってくる色のプレビューが表示されます。
- 消えかけのかたまりに、後から同じ色をつなぐと **消える時間がリセット** されます。
- 連続で消すと **チェーン** が増え、スコア倍率が上がります（1連鎖=1倍、2連鎖=2倍…）。

## タイムリミット

- 画面上部のバーが **残り30秒** を示します。バーが無くなるとゲームオーバーです。
- **5個以上つながった瞬間** にパネル1枚あたり0.5秒回復します（上限30秒）。
- 1分ごとにレベルが上がり、バーの消費速度が速くなります（最大レベル31で約5秒で空に）。

## スコア・ランキング

- ゲームオーバー時のスコアはブラウザの **localStorage** に保存され、ベスト10が表示されます。
- 画面上部に **SEED** 値（8桁16進数）が表示されます。毎回異なるシードで乱数が初期化されます。

## セットアップ / 起動

```bash
npm install      # 依存関係のインストール
npm run build    # TypeScript を dist/app.js にコンパイル
npm start        # lite-server で起動（ブラウザが開きます）
```

開発中は `npm run watch` で TypeScript の自動コンパイルができます。

## 技術スタック

- TypeScript（`src/app.ts` → `dist/app.js`）
- 素の HTML / CSS（フレームワークなし、DOM ベース描画）
- Web Audio API（効果音をコード内生成）
- lite-server（開発サーバ）
