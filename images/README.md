# パネル画像

盤面の5色パネルに使う画像です。以下のファイル名で配置してください（正方形PNG推奨）。

| ファイル名     | 色     |
| -------------- | ------ |
| `red.png`      | 赤     |
| `blue.png`     | 青     |
| `green.png`    | 緑     |
| `yellow.png`   | 黄     |
| `purple.png`   | 紫     |

- 現在置かれているのは動作確認用の**プレースホルダー画像**です。好きな画像に差し替えてください。
- 画像は `background-size: cover` で表示されるため、正方形だと崩れません。
- 画像の読み込みに失敗した場合は、従来の単色（CSS変数 `--c-*`）にフォールバックします。
